# Installation
- Unzip config files to RtCW\Main or RtCW\rtcwpro 
- edit `autoexec_allies.cfg` and `autoexec_axis.cfg` to change default Bindings

### autoexec_timer
- Press P when enemy spawns

### Spawnpoint toggle
- Press F5 to swap between spawnpoints
